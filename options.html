H2R - Translate

English → Bengali translation in your browser, powered by AI.

Your key is stored only in your browser using Chrome's extension storage and is sent directly to the provider you select. It is never shared with anyone else.

Leave blank to use the recommended default for the selected provider.

How to use

  1. Select any English word or sentence on a web page.
  2. Right-click and choose “H2R - Translate” (or “H2R - Lookup”).
  3. A tooltip shows the Bengali translation, the English meaning, and example usage.
